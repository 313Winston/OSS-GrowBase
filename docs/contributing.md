# CONTRIBUTING.md

```markdown
# Beitrag leisten zu OSS-GrowBase

Vielen Dank, dass du mithelfen möchtest, GrowBase Pro zu verbessern! Hier sind einige Richtlinien, die dir beim Einstieg helfen.

## 💡 Ideen und Vorschläge
- Teile neue Ideen im Discord-Kanal `#ideen-und-vorschläge` oder als Issue auf GitHub.

## 🐛 Fehler melden
1. Öffne ein Issue in [GitHub Issues](https://github.com/313Winston/growbase-pro/issues).
2. Beschreibe das Problem klar und prägnant.
3. Füge relevante Informationen wie Logs oder Screenshots hinzu.

## 🔧 Code beitragen
1. **Forke das Repository** auf GitHub.
2. **Clonen**:
   ```bash
   git clone https://github.com/313Winston/growbase-pro.git
   ```
3. Erstelle einen neuen Branch:
   ```bash
   git checkout -b feature/MEINE-FUNKTION
   ```
4. Implementiere deine Änderungen und teste sie.
5. **Pull Request erstellen**:
   - Beschreibe deine Änderungen im Pull Request.
   - Verlinke ggf. das zugehörige Issue.

## 🔍 Code-Richtlinien
- Schreibe sauberen, gut dokumentierten Code.
- Halte dich an bestehende Code-Formate und Standards (z. B. PEP 8 für Python).

## 🌱 Danke!
Jeder Beitrag hilft dabei, OSS-GrowBase besser zu machen. 💚

```
